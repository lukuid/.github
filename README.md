# LukuID

> Hardware-backed forensic evidence with offline verification.

LukuID builds secure devices and software for generating tamper-resistant, cryptographically verifiable records from the physical world.

---

## What we build

- Cryptographically attested hardware devices for trusted data capture  
- The `.luku` evidence format (portable, offline-verifiable)  
- Open SDKs (Rust, JavaScript, Python, Swift, Kotlin)  
- CLI tools for inspection and verification  
- A trust model based on device attestation and signed continuity  

---

## What it enables

Independent verification of:

- RFID / NFC scans  
- Environmental telemetry  
- Custody and handoff events  
- Attached documents and third-party endorsements  

All without requiring access to a central API.
